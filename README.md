# generator-weiqi-koa2-react-server

cli

* koa2
* react
* server-render
* css modules

> install

```
npm install yo generator-weiqi-koa2-react-server -g
```

> start

```
yo weiqi-koa2-react-server
cd yourPath
npm install
npm run dev
```
